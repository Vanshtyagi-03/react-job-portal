# 🚀 React Job Portal

A modern **Job Portal Web Application** built using **React.js + Vite**.

# 🚀 React Job Portal

[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge&logo=vercel)](https://react-job-portal-nine.vercel.app)

## 🔥 Features

- Browse 50+ jobs
- Save jobs ⭐
- Apply for jobs with form
- Resume upload
- Company logos
- Top companies section
- Trending jobs
- Admin Dashboard 📊
- Smart job search
- Dark mode 🌙

## 🛠 Tech Stack

- React.js
- Vite
- JavaScript
- Bootstrap
- LocalStorage

## 📸 Pages

- Home Page
- Job Listings
- Job Details
- Apply Job Form
- Saved Jobs
- Admin Dashboard
